## Usage
<Code code="yarn config get <name>" lang="bash" />

## Examples
<p>Print a simple configuration setting
:</p>
<Code code="yarn config get yarnPath" lang="bash" />

<p>Print a complex configuration setting
:</p>
<Code code="yarn config get packageExtensions" lang="bash" />

<p>Print a nested field from the configuration
:</p>
<Code code="yarn config get 'npmScopes[&quot;my-company&quot;].npmRegistryServer'" lang="bash" />

<p>Print a token from the configuration
:</p>
<Code code="yarn config get npmAuthToken --no-redacted" lang="bash" />

<p>Print a configuration setting as JSON
:</p>
<Code code="yarn config get packageExtensions --json" lang="bash" />

## Details
This command will print a configuration setting.

Secrets (such as tokens) will be redacted from the output by default. If this 
behavior isn't desired, set the `--no-redacted` to get the untransformed value.

## Options
| Definition | Description |
| ---------- | ----------- |
| <h4 id="options-why" className="header-code"><code className="language-text">--why</code></h4> | Print the explanation for why a setting has its value |
| <h4 id="options-json" className="header-code"><code className="language-text">--json</code></h4> | Format the output as an NDJSON stream |
| <h4 id="options-no-redacted" className="header-code"><code className="language-text">--no-redacted</code></h4> | Don't redact secrets (such as tokens) from the output |
