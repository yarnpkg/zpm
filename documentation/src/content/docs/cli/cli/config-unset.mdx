</div>

## Usage
<Code code="yarn config unset <name>" lang="bash" />

## Examples
<p>Unset a simple configuration setting
:</p>
<Code code="yarn config unset initScope" lang="bash" />

<p>Unset a complex configuration setting
:</p>
<Code code="yarn config unset packageExtensions" lang="bash" />

<p>Unset a nested configuration setting
:</p>
<Code code="yarn config unset npmScopes.company.npmRegistryServer" lang="bash" />

## Details
This command will unset a configuration setting.

## Options
| Definition | Description |
| ---------- | ----------- |
| <h4 id="options-H%2C-home" className="header-code"><code className="language-text">-H,--home</code></h4> | Update the home configuration instead of the project configuration |
