## Usage
<Code code="yarn plugin import from sources <name>" lang="bash" />

## Examples
<p>Build and activate the "@yarnpkg/plugin-exec" plugin
:</p>
<Code code="yarn plugin import from sources @yarnpkg/plugin-exec" lang="bash" />

<p>Build and activate the "@yarnpkg/plugin-exec" plugin (shorthand)
:</p>
<Code code="yarn plugin import from sources exec" lang="bash" />

## Details
This command clones the Yarn repository into a temporary folder, builds the 
specified contrib plugin and updates the configuration to reference it in 
further CLI invocations.

The plugins can be referenced by their short name if sourced from the official 
Yarn repository.
