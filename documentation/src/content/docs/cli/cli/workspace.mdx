## Usage
<Code code="yarn workspace <workspaceName> <commandName> ..." lang="bash" />

## Examples
<p>Add a package to a single workspace
:</p>
<Code code="yarn workspace components add -D react" lang="bash" />

<p>Run build script on a single workspace
:</p>
<Code code="yarn workspace components run build" lang="bash" />

## Details
This command will run a given sub-command on a single workspace.
