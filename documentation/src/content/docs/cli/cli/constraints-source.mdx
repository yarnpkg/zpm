</div>

## Usage
<Code code="yarn constraints source" lang="bash" />

## Examples
<p>Prints the source code
:</p>
<Code code="yarn constraints source" lang="bash" />

<p>Print the source code and the fact database
:</p>
<Code code="yarn constraints source -v" lang="bash" />

## Details
This command will print the Prolog source code used by the constraints engine. 
Adding the `-v,--verbose` flag will print the *full* source code, including the 
fact database automatically compiled from the workspace manifests.

## Options
| Definition | Description |
| ---------- | ----------- |
| <h4 id="options-v%2C-verbose" className="header-code"><code className="language-text">-v,--verbose</code></h4> | Also print the fact database automatically compiled from the workspace manifests |
