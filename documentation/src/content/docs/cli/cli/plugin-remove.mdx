## Usage
<Code code="yarn plugin remove <name>" lang="bash" />

## Examples
<p>Remove a plugin imported from the Yarn repository
:</p>
<Code code="yarn plugin remove @yarnpkg/plugin-typescript" lang="bash" />

<p>Remove a plugin imported from a local file
:</p>
<Code code="yarn plugin remove my-local-plugin" lang="bash" />

## Details
This command deletes the specified plugin from the .yarn/plugins folder and 
removes it from the configuration.

**Note:** The plugins have to be referenced by their name property, which can be 
obtained using the `yarn plugin runtime` command. Shorthands are not allowed.
