## Usage
<Code code="yarn npm tag remove <package> <tag>" lang="bash" />

## Examples
<p>Remove the `beta` tag from package `my-pkg`
:</p>
<Code code="yarn npm tag remove my-pkg beta" lang="bash" />

## Details
This command will remove a tag from a package from the npm registry.
