## Usage
<Code code="yarn explain [code]" lang="bash" />

## Examples
<p>Explain an error code
:</p>
<Code code="yarn explain YN0006" lang="bash" />

<p>List all error codes
:</p>
<Code code="yarn explain" lang="bash" />

## Details
When the code argument is specified, this command prints its name and its 
details.

When used without arguments, this command lists all error codes and their names.
