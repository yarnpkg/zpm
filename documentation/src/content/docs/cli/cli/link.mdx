</div>

## Usage
<Code code="yarn link ..." lang="bash" />

## Examples
<p>Register one or more remote workspaces for use in the current project
:</p>
<Code code="yarn link ~/ts-loader ~/jest" lang="bash" />

<p>Register all workspaces from a remote project for use in the current project
:</p>
<Code code="yarn link ~/jest --all" lang="bash" />

## Details
This command will set a new `resolutions` field in the project-level manifest 
and point it to the workspace at the specified location (even if part of another 
project).

## Options
| Definition | Description |
| ---------- | ----------- |
| <h4 id="options-A%2C-all" className="header-code"><code className="language-text">-A,--all</code></h4> | Link all workspaces belonging to the target projects to the current one |
| <h4 id="options-p%2C-private" className="header-code"><code className="language-text">-p,--private</code></h4> | Also link private workspaces belonging to the target projects to the current one |
| <h4 id="options-r%2C-relative" className="header-code"><code className="language-text">-r,--relative</code></h4> | Link workspaces using relative paths instead of absolute paths |
