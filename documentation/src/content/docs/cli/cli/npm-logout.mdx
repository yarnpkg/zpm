## Usage
<Code code="yarn npm logout" lang="bash" />

## Examples
<p>Logout of the default registry
:</p>
<Code code="yarn npm logout" lang="bash" />

<p>Logout of the @my-scope scope
:</p>
<Code code="yarn npm logout --scope my-scope" lang="bash" />

<p>Logout of the publish registry for the current package
:</p>
<Code code="yarn npm logout --publish" lang="bash" />

<p>Logout of all registries
:</p>
<Code code="yarn npm logout --all" lang="bash" />

## Details
This command will log you out by modifying your local configuration (in your 
home folder, never in the project itself) to delete all credentials linked to a 
registry.

Adding the `-s,--scope` flag will cause the deletion to be done against whatever 
registry is configured for the associated scope (see also `npmScopes`).

Adding the `--publish` flag will cause the deletion to be done against the 
registry used when publishing the package (see also `publishConfig.registry` and 
`npmPublishRegistry`).

Adding the `-A,--all` flag will cause the deletion to be done against all 
registries and scopes.
