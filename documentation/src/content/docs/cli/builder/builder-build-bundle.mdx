## Usage
<Code code="yarn builder build bundle" lang="bash" />

## Examples
<p>Build the local bundle
:</p>
<Code code="yarn builder build bundle" lang="bash" />

<p>Build the local development bundle
:</p>
<Code code="yarn builder build bundle --no-minify" lang="bash" />

## Details
This command builds the local bundle - the Yarn binary file that is installed in 
projects.

For more details about the build process, please consult the `@yarnpkg/builder` 
README: 
https://github.com/yarnpkg/berry/blob/HEAD/packages/yarnpkg-builder/README.md.
